# 미루루 굿즈샵 - UI Spec

## 경로

`/goods/miruru`

## 페이지 목적

- 미루루 관련 보이스팩 및 굿즈 판매
- 브랜드 정체성 전달
- 구매 경험 최적화

⭐ **본 페이지는 1차 MVP의 핵심 페이지입니다.**

## 컨셉 및 테마

### 디자인 컨셉
- **포근하고 다정한 동물의 숲**
- 말랑말랑한 파스텔 하늘색 (#A8D5E2, #E3F2FD 등)
- 부드러운 모서리, 귀여운 아이콘
- 따뜻하고 친근한 분위기

### 색상 팔레트
- 메인: 파스텔 하늘색
- 서브: 크림색, 연한 핑크
- 텍스트: 부드러운 회색

## 레이아웃 구조

### 1. Hero Section (메인 굿즈 홍보)

**목적**: 최신/인기 굿즈 홍보

**구성**:
- 메인 굿즈 이미지/배너
- 굿즈 제목 및 간단한 설명
- "구매하기" CTA 버튼
- 자동 슬라이드 (선택, 여러 굿즈 홍보 시)

**레이아웃**:
- 전체 너비 또는 중앙 정렬
- 모바일: 세로 배치
- 데스크톱: 좌우 배치 (이미지 | 텍스트)

### 2. Voice Packs Section (보이스팩)

**목적**: 보이스팩 판매 (주력 상품)

**디자인 컨셉**: **CD 플레이어 느낌**

**구성**:
- 섹션 제목: "Voice Packs" 또는 "보이스팩"
- 보이스팩 카드 그리드
- 각 카드:
  - 썸네일 이미지 (CD 커버 느낌)
  - 보이스팩 제목
  - 가격
  - 샘플 재생 버튼 (중요!)
  - "구매하기" 버튼

**샘플 재생 기능**:
- 클릭 시 오디오 플레이어 팝업 또는 인라인 재생
- 재생/일시정지/정지 컨트롤
- 진행 바 표시

**레이아웃**:
- 모바일: 1-2열 그리드
- 데스크톱: 3-4열 그리드
- CD 플레이어 느낌의 카드 디자인 (원형 또는 정사각형)

### 3. Physical Goods Section (실물 굿즈)

**목적**: 실물 굿즈 판매

**구성**:
- 섹션 제목: "Goods" 또는 "실물 굿즈"
- 굿즈 카드 그리드
- 각 카드:
  - 상품 이미지
  - 상품 이름
  - 가격
  - "구매하기" 버튼

**레이아웃**:
- 모바일: 1-2열 그리드
- 데스크톱: 3-4열 그리드

### 4. Shop Info Section (안내)

**목적**: 구매 관련 안내

**구성**:
- 배송 정책
- 제작 기간 안내
- 환불 정책
- FAQ (선택)

**레이아웃**:
- 접을 수 있는 아코디언 형태 또는 탭 형태

## 상태 관리

### 상품 데이터
- `voicePacks`: 보이스팩 목록
- `physicalGoods`: 실물 굿즈 목록
- `isLoading`: 로딩 상태
- `error`: 에러 상태

### 오디오 재생
- `currentPlaying`: 현재 재생 중인 오디오 ID
- `isPlaying`: 재생 상태
- `progress`: 재생 진행도

## 데이터 구조

```typescript
interface VoicePack {
  id: string
  title: string
  description: string
  price: number
  thumbnail: string
  sampleAudioUrl: string
  digitalFileUrl: string
  isActive: boolean
}

interface PhysicalGoods {
  id: string
  name: string
  description: string
  price: number
  images: string[]
  isActive: boolean
}
```

## 인터랙션

### 보이스팩 샘플 재생
1. 샘플 재생 버튼 클릭
2. 오디오 플레이어 표시
3. 재생/일시정지 토글
4. 다른 샘플 클릭 시 이전 재생 정지

### 구매하기
1. "구매하기" 버튼 클릭
2. 로그인 확인 (비로그인 시 로그인 페이지로 리다이렉트)
3. 상품 상세 페이지 또는 바로 장바구니/주문 페이지로 이동

## 디자인 가이드

### 카드 디자인
- 부드러운 모서리 (border-radius: 16px)
- 파스텔 하늘색 테두리 또는 배경
- 호버 시 살짝 떠오르는 효과

### 버튼
- 메인 버튼: 파스텔 하늘색 배경
- 텍스트: 흰색 또는 부드러운 회색
- 부드러운 그림자

### 아이콘
- 재생 버튼: 귀여운 스타일
- 장바구니, 하트 등: 부드러운 라인

### 타이포그래피
- 제목: 부드러운 산세리프
- 본문: 가독성 좋은 폰트

## 반응형

### 모바일 (< 768px)
- 1-2열 그리드
- 카드 간격: 16px
- Hero 이미지 전체 너비

### 태블릿 (768px ~ 1024px)
- 2-3열 그리드
- 카드 간격: 24px

### 데스크톱 (> 1024px)
- 3-4열 그리드
- 카드 간격: 32px
- 최대 너비 제한 (1200px)

## 접근성

- 오디오 플레이어 키보드 컨트롤
- 재생/일시정지 상태 명확히 표시
- 이미지 alt 텍스트
- 버튼 명확한 레이블

## 성능

- 이미지 최적화 (Next.js Image)
- 오디오 lazy loading
- 초기 로딩 속도 우선
- 샘플 오디오 크기 최적화 (30초 이내 짧은 샘플)

## 1차 MVP 제외 기능

- 후기 시스템
- 추천 알고리즘
- 재생 플레이리스트 저장
- 위시리스트
- 장바구니 (바로 주문으로 단순화)

## API 연동

- 보이스팩 목록: `GET /api/products?artistId={miruru}&type=VOICE_PACK`
- 실물 굿즈 목록: `GET /api/products?artistId={miruru}&type=PHYSICAL_GOODS`
- 주문 생성: `POST /api/orders`

## 참고사항

- 보이스팩 구매가 MVP의 생존 조건
- 샘플 재생 기능은 필수
- 단순하고 직관적인 구매 플로우
- 미루루 브랜드 정체성 명확히 전달
